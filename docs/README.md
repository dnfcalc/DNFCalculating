安装项目依赖<br>
> pip install -i https://pypi.mirrors.ustc.edu.cn/simple/ -r requirements.txt

## macOS下运行方式

&#160; &#160; &#160; &#160; 直接在`Finder`中双击`run_mac.command`即可运行。

&#160; &#160; &#160; &#160; 或者在`终端(Terminal)`中，执行命令`python3.8 main.py`即可运行。
